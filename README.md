# SKHU-APP

성공회대학교 종합정보시스템 모바일 앱

## 개발자

- 추건우 - 소프트웨어공학과 201534028

## 스택

- react-native
- redux
- redux-observable
- rxjs
- react-router-redux
- react-router-native

## 실행

- expo 기반의 환경

```bash
  > yarn start 

  (QR코드가 나타납니다. expo 앱을 통하여 실행됩니다.)
```
```bash
  > yarn ios 
  
  (Android SDK가 있다면 가상기기를 통해 실행됩니다.)
```
```bash
  > yarn andriod 
  
  (XCode가 있다면 가상기기를 통해 실행됩니다.)
```
